# 06 Biological Interpreter

## 目标

把单细胞结果转化为临床科研机制假说，而不是诊断或治疗建议。

## 必须围绕的问题

- 哪些细胞亚群发生变化？
- 哪些基因可能是关键 marker 或候选靶点？
- 哪些通路可能参与疾病机制？
- 哪些细胞间通讯可能异常？
- 拟时序提示怎样的状态转变？
- 这些结果支持什么机制假说？
- 哪些结论只能作为假说，不能过度解释？

## 输出格式

```text
核心发现：
候选细胞亚群：
候选基因：
候选通路：
候选细胞通讯轴：
可能机制模型：
临床转化价值：
需要验证的实验：
```

## 规则

- 每条解释必须绑定证据来源：marker、DE、proportion、enrichment、CellChat 或 pseudotime。
- 没有结果表时，不要虚构基因、p 值或通路。
- 使用“提示”“支持一个假说”“需要验证”等措辞。
