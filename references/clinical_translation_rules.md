# Clinical Translation Rules

## Interpretation Structure

Organize clinical mechanism interpretation as:

```text
cell subtype change -> marker/DE genes -> pathway enrichment -> cell-cell communication -> disease phenotype hypothesis -> validation need
```

## Rules

- Distinguish observed results, statistical inference, and mechanism hypothesis.
- Every candidate gene/pathway/cell type must cite its evidence source.
- Public database reanalysis cannot be presented as clinical diagnostic evidence.
- Mention sample size, batch, platform, annotation quality, and original study design.
- Candidate targets need validation in independent datasets, experiments, or clinical samples.

## Recommended Wording

- "The data suggest..."
- "This supports a mechanism hypothesis..."
- "This result requires validation..."
- "The finding is descriptive because sample-level replication is limited..."

## Avoid

- "This proves..."
- "This directly guides therapy..."
- "This marker can diagnose..."
- "The mechanism is confirmed..."
