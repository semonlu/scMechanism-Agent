# Manuscript-Style Report: GSE223751 Single-Cell Reanalysis

## Scope

- Dataset: Single-cell RNA sequencing reveals cellular and molecular heterogeneity in fibrocartilaginous enthesis formation
- Accession: GSE223751
- Organism: Mus musculus
- Tissue/context: rotator cuff fibrocartilaginous enthesis
- Scope note: Shoulder-region enthesis developmental single-cell example; not a frozen-shoulder or shoulder periarthritis case-control cohort.

## Methods

Public processed single-cell RNA-seq matrices were analyzed locally using Seurat V5, SingleR, marker-support scoring, cluster marker detection, GO/KEGG enrichment, CellChat, and Monocle3. Sample-level 10x matrices were imported into Seurat, merged with sample metadata, filtered after QC inspection, normalized, scaled, reduced by PCA/UMAP, and clustered. Reference-assisted annotation used SingleR, and marker-support labels were computed from curated marker sets. Cluster markers were used for GO/KEGG enrichment. Cell-cell communication was inferred with CellChat. Trajectory analysis used Monocle3 with reviewed Seurat UMAP coordinates, a marker-supported multi-state trajectory input, graph learning, and an E15 root rule.

## Results

### Data Processing And QC

The processed Seurat object retained 46,955 cells across 7 sample entries.
Retained cells by stage were: E15 11,043; P1 7,404; P14 3,377; P28 16,172; P7 8,959.

### Cell Annotation

SingleR annotation identified the following major labels: Fibroblasts 25,493 (54.3%); Erythrocytes 12,261 (26.1%); B cells 6,651 (14.2%); Endothelial cells 1,377 (2.9%); Cardiomyocytes 1,173 (2.5%).
These labels should be reviewed with marker evidence and tissue context before biological claims are finalized.

### Marker Enrichment

Top GO biological-process terms among selected marker genes included:
- muscle system process (adjusted p=4.78262837804851e-10, genes=17)
- homeostasis of number of cells (adjusted p=2.23409332519696e-05, genes=12)
- striated muscle cell differentiation (adjusted p=2.23409332519696e-05, genes=11)
- striated muscle cell development (adjusted p=2.23409332519696e-05, genes=9)
- muscle cell differentiation (adjusted p=3.42823146044387e-05, genes=12)

Top KEGG terms included:
- Cytoskeleton in muscle cells (adjusted p=3.20570489831851e-09, genes=13)
- Malaria (adjusted p=1.77128848247043e-05, genes=6)
- Primary immunodeficiency (adjusted p=1.77128848247043e-05, genes=5)
- Motor proteins (adjusted p=7.49832995903589e-05, genes=8)
- African trypanosomiasis (adjusted p=0.000422856793365406, genes=4)

### Cell-Cell Communication

CellChat inferred ligand-receptor interactions, including the highest-probability exported interactions:
- Fibroblasts -> Erythrocytes: Ptn / Ncl (PTN, prob=0.0942702736101875)
- Fibroblasts -> Cardiomyocytes: Ptn / Ncl (PTN, prob=0.0770977229842359)
- Fibroblasts -> Fibroblasts: Ptn / Ncl (PTN, prob=0.0696028548876548)
- Fibroblasts -> Endothelial cells: Ptn / Ncl (PTN, prob=0.0656353876814986)
- Fibroblasts -> B cells: Ptn / Ncl (PTN, prob=0.064223174488382)
These interactions are database-based computational hypotheses, not experimental proof of signaling.

### Pseudotime And Trajectory

Monocle3 trajectory analysis used 5,000 cells with 4,537 unique finite pseudotime values. Trajectory input states were: tenocyte_fibroblast 3,126; chondrocyte 820; cycling 762; osteoblast 292.
Median pseudotime by state was: cycling median 0.0553; tenocyte_fibroblast median 0.182; osteoblast median 1.41; chondrocyte median 2.26.
The trajectory should be interpreted as relative transcriptional ordering across marker-supported states, not measured chronological time or causal proof.

## Mechanism Hypothesis

Together, marker enrichment, inferred communication, and trajectory ordering support an exploratory hypothesis that mesenchymal, chondrogenic/osteogenic, and proliferative states are organized along a developmental enthesis-associated continuum. This hypothesis is dataset- and annotation-dependent and requires validation in independent shoulder/enthesis or disease-relevant cohorts.

## Limitations

- This is a public processed-data reanalysis and does not replace a prospectively designed clinical study.
- The example dataset is shoulder-region rotator cuff enthesis biology, not a direct frozen-shoulder case-control cohort.
- SingleR and marker-support annotations require manual review and independent validation.
- Enrichment and CellChat analyses depend on database coverage and exported marker/interaction tables.
- Pseudotime is a relative ordering inferred from expression similarity and graph structure, not a direct time measurement.
- Mechanism hypotheses should be validated in independent cohorts and targeted experiments.

## Validation Suggestions

- Review marker expression for each trajectory state and adjust state labels if tissue expertise suggests alternatives.
- Repeat key marker, enrichment, CellChat, and pseudotime analyses in an independent disease- or injury-relevant dataset.
- Validate candidate pathways and ligand-receptor interactions experimentally or with orthogonal spatial/protein evidence.
