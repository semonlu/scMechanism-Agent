# Results Template

## Data Availability And QC

Describe input files, sample metadata, retained cells, and QC decisions. Do not invent cell counts if not provided.

## Cell Clustering And Annotation

Describe clusters and cell-type labels with marker evidence. Mark uncertain labels explicitly.

## Group Comparison

Describe disease/control or treatment comparisons only if sample-level metadata and statistical outputs are available.

## Enrichment, Communication, And Trajectory

Separate computed results from mechanism hypotheses. State database/tool assumptions.

## Limitations

Mention sample size, batch effects, annotation uncertainty, processed-data limitations, and validation needs.
